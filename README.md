# RL Playground — LunarLander Baselines

A ready-to-use reinforcement-learning playground for **LunarLander** (both
discrete and continuous action spaces) using
[Stable-Baselines3](https://stable-baselines3.readthedocs.io/) and
[Gymnasium](https://gymnasium.farama.org/).

Train, evaluate, and compare six baseline algorithms side-by-side with a single
command.

---

## Algorithms

| Algorithm | Action Space | Environment |
|-----------|-------------|-------------|
| **DQN**   | Discrete    | `LunarLander-v3` |
| **PPO**   | Discrete    | `LunarLander-v3` |
| **A2C**   | Discrete    | `LunarLander-v3` |
| **SAC**   | Continuous  | `LunarLanderContinuous-v3` |
| **DDPG**  | Continuous  | `LunarLanderContinuous-v3` |
| **TD3**   | Continuous  | `LunarLanderContinuous-v3` |

The environment is considered **solved** when an agent achieves a mean reward
≥ 200 over 100 consecutive episodes.

## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

> **Note:** `gymnasium[box2d]` requires `swig`. Install it with
> `sudo apt-get install swig` (Linux) or `brew install swig` (macOS).

### 2. Train all algorithms

```bash
python train_all.py
```

This trains every algorithm with its default hyperparameters (see
[`configs.py`](configs.py)) and saves models + episode logs under
`results/<ALGO>/`.

To train a **single** algorithm:

```bash
python train.py --algo PPO
python train.py --algo DQN --timesteps 300000 --seed 0
python train.py --algo SAC
```

For a quick smoke-test with fewer steps:

```bash
python train_all.py --timesteps 50000
```

### 3. Compare results

```bash
python compare.py
```

Generates `results/comparison.png` — a side-by-side plot of the training
reward curves for all algorithms that have been trained.

Options:

```bash
python compare.py --algos DQN PPO A2C   # subset of algorithms
python compare.py --window 50           # wider smoothing window
python compare.py --output my_plot.png  # custom output path
python compare.py --no_save             # show interactively
```

### 4. Evaluate a trained agent

```bash
python evaluate.py --algo PPO            # default: best_model, 10 episodes
python evaluate.py --algo DQN --episodes 20
python evaluate.py --algo SAC --render   # requires a display
python evaluate.py --algo TD3 --model_path results/TD3/models/final_model
```

    ├── TD3/

## Customising Hyperparameters

Edit [`configs.py`](configs.py).  Each entry follows this schema:

```python
"PPO": {
    "env": "LunarLander-v3",
    "timesteps": 1_000_000,
    "hyperparams": {
        "policy": "MlpPolicy",
        "learning_rate": 3e-4,
        ...
    },
},
```

All `hyperparams` keys (except `policy`) are passed directly to the
Stable-Baselines3 constructor, so any SB3-supported argument is accepted.
